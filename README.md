# Appointment Management System

A Spring Boot–based application designed to manage doctor–patient appointments, 
including availability, scheduling, and appointment status management.

- Doctor, patient and admin user roles
- Department management
- Doctors can create daily availability
- Patients can book apointments based on availability
- Apointment Status Management
- Manual DTO-Entity Mapping
- PostgreSQL database integration

  ## Technologies Used
- Java 21
- Spring Boot
- Spring Data JPA
- Spring Security
- PostgreSQL
- Lombok
- Maven

  ## How to Run

1. Clone the repository:
   git clone https://github.com/.../AppointmentManagementSystem.git

2. Create a PostgreSQL database:
   CREATE DATABASE appointment_db;

3. Configure database credentials in application.properties

4. Run the project:
   mvn spring-boot:run
